# phpmybdayRSVP
A birthday invitation website built with php.

## Features
- [x] Main party info landing page
- [x] Online rsvp form
- [x] Back-end, password protected admin login
- [x] Admin can view and filter rsvp responses, which are stored and pulled from database.
- [x] Saves paper and simplifies the rsvp process.
